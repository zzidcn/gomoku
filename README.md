# 五子棋 Gomoku v2.0.0

一个用 Python + tkinter 实现的五子棋桌面游戏，支持 AI 对战、音效、多主题换肤。

## 新功能 v2.0.0

- 🤖 **AI 对战**：内置 Minimax + Alpha-Beta 剪枝算法，支持 AI 执黑/执白
- 🎵 **音效**：落子、胜利、悔棋音效（pygame 合成，无需音频文件）
- 🎨 **5 套主题**：经典木纹 / 翠竹青 / 深夜蓝 / 樱花粉 / 墨金
- ↩️ **智能悔棋**：AI 模式下自动撤回两步

## 功能特性

- 🎮 15×15 标准棋盘
- ⚫⚪ 双人对战 / 人机对战（三种模式）
- 🏆 自动判断胜负（五子连珠）
- 👁️ 鼠标悬停预览落子位置
- 🔴 最后落子位置标记
- 🔊 音效开关

## 运行方式

### 方式一：直接运行 exe（推荐）

从 [Releases](../../releases) 下载 `gomoku.exe`，双击运行，无需安装任何环境。

> 若被系统拦截：右键 → 属性 → 勾选「解除锁定」→ 确定

### 方式二：源码运行

```bash
pip install pygame numpy
python gomoku.py
```

## 游戏模式

| 模式 | 说明 |
|------|------|
| 双人对战 | 两人轮流在同一台电脑上对弈 |
| AI 执白 | 玩家执黑先手，AI 执白应对 |
| AI 执黑 | AI 执黑先手，玩家执白应对 |
